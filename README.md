## ✨ 功能

- [x] 云仔主体
- [ ] js template (用于云仔插件开发)
- [x] csv-逍遥插件 （完成等release）
- [x] 喵喵插件 （完成等release）



## 🖥 依赖

## 📦 安装与更新

1. 安装koishi
2. 安装redis
2. 插件市场搜索 @Kuroshio/genshin， 选择需要的版本
3. 配置插件就可
